# BigQuery Reference

Welcome to the BigQuery reference guide. This page contains information about how to interact with BigQuery within the Fluency platform.

## Overview

Google BigQuery is a serverless, highly scalable, and cost-effective cloud data warehouse.

## Key Features

- **SQL Interface**: Use standard SQL to query your data.
- **Scalability**: Handle datasets of any size.
- **Speed**: Query gigabytes to petabytes of data in seconds.

## Usage in Fluency

You can use BigQuery datasets as a source for your knowledge vault or as a backend for complex data analysis.

### Connecting to BigQuery

To connect your BigQuery instance:
1. Navigate to Settings.
2. Select Integrations.
3. Choose BigQuery and follow the authentication steps.
