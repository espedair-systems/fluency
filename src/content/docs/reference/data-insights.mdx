# Data Insights Reference

Data Insights provides automated discoveries, statistical summaries, and AI-driven analysis of your datasets.

## Overview

The Insights engine scans your data to identify trends, anomalies, and key performance indicators without requiring manual query writing.

## Key Features

- **Automated Discovery**: Automatically detects correlations and significant patterns in your data.
- **Natural Language Querying**: Ask questions about your data in plain English.
- **Visual Summaries**: Generate charts and dashboards with a single click.

## Core Capabilities

### Predictive Analytics
Insights uses historical data to forecast future trends and identify potential risks.

### Anomaly Detection
Get alerted when data points fall outside of expected ranges or when sudden shifts in metrics occur.

### Data Profiling
Quickly understand the shape, distribution, and quality of any dataset in your vault.

## Usage

Access insights by navigating to any dataset and selecting the **Insights** tab.
